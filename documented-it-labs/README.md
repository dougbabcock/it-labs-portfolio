# Documented IT Labs

Skimmable summaries and source lab documents for Doug Babcock's IT support / systems administration lab portfolio.

## Labs

1. [Virtualized Windows Environment](Virtualized_Windows_Environment_Documented_Lab.md)
   - Built a small VirtualBox lab with Windows Server 2022 and Windows 11 workstations.
   - Configured NAT networking, cloned a workstation, assigned/verified IP connectivity, and tested internal/internet reachability.

2. [Windows Group Policy Security](Windows_Group_Policy_Security_Documented_Lab.md)
   - Implemented domain security controls through Group Policy.
   - Configured password/account lockout policy, a domain logon security message, corporate desktop branding, and verified deployment with `gpupdate` / `gpresult`.

3. [Software Installation and Remote User Support](Software_Installation_Remote_User_Support_Documented_Lab.md)
   - Created a centralized software repository and supported a user workstation remotely.
   - Downloaded approved software, shared installers from a server, connected to a client over Remote Desktop, installed applications, and verified successful operation.
